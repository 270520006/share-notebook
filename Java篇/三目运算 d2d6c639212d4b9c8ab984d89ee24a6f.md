# 三目运算

## 示例代码:

```java
// 三目
<td th:text="${vehicleRecord.isTemp==1?'临时车':'已购车位'}"></td>

//多条件三目
<td th:text=
"${vehicleRecord.parkingAreaId==1?'A区':(vehicleRecord.parkingAreaId==2?'B区'(vehicleRecord.parkingAreaId==3?'C区':'D区'))}">
                            </td>
```

```java
//三木运算解析
// 如果A为空结果就为B 反之
A==Null?B:C

//多条件三木运算解析

A==Null?B:(A==1?C:(A==2?D:F))
```
