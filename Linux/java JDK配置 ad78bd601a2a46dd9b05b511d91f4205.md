# java JDK配置

> jdk的安装
1.找到安装包的位置
2.查看是否已安装
java -version
3.是否存在已安装过的包
rpm -qa | grep -i jdk
4.正式安装
rpm -ivh 包名
=================================
环境变量配置
5.环境变量文件 /etc/profile 用root编辑，vi或者图形化修改都可以。
export JAVA_HOME=/opt/java/jdk1.8.0_271
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
