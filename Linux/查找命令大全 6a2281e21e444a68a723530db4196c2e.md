# 查找命令大全

### grep

```bash
grep 命令最大功能是在一堆文件中查找一个特定的字符串
grep money test.txt
以上命令在 test.txt中查找money这个字符串,grep 查找是区分大小的
```

### **find**

```bash
基本格式: find / -name fileName

--------------------------------------------------------------

精准定位+快速搜索: find /dirpath -name fileName

基本格式: find /etc -name my.cnf

在etc 目录下查询名称为my.cnf 的文件的文件路径

--------------------------------------------------------------

模糊查询：find / dirpath -name '*fileName*'

根据部分文件名查找方法：

这个方法和在WINDOWS中查找已知的文件名方法是一样的。不过在Linux中根据部分文件名查找文件的方法要比在WINDOWS中的同类查找方法要强大得多

基本格式: find /etc -name '*my*'

在etc 目录下查询名称包含 'my' 的文件的文件路径

--------------------------------------------------------------

如果只知道某个文件的大小，修改日期等特征也可以使用“find”命令查找出来，这和WINDOWS系统中的"搜索"功能是基本相同的。

通过文件大小查询:
基本格式: find /dirpath -size fileSize

如:find /etc -size 1066c (注：'c' 表示 byte)

--------------------------------------------------------------

如果我们连这个文件的具体大小都不知道，那么在Linux中还可以进行模糊查找方式来解决。
例如我们输入 find / -size +10000000c 这个命令，则标明我们指定系统在根目录中查找出大于 10000000 字节的文件并显示出来。
命令中的"＋"(加号)是表示要求系统只列出大于指定大小的文件，而使用 "**-"(减号)** 则表示要求系统列出小于指定大小的文件。

基本格式: find /var/tomcat -size +4000000c 在tomcat 目录下查询大于4000000bytes的文件
```

### which

```bash
基本格式: which (command)

依次从path 环境变量所列的目录中找到 command 的位置,并显示完整路径的名称。 在找到第一个符合条件的程序文件时,就立即停止搜索,省略其余
未搜索目录
```

### whereis

```bash
基本格式: whereis [option] name

找出特定程序的可执行文件、源代码文件以及manpage的路径。你所提供的name 会被先除去前置的路径以及任何 .ext 形式的拓展名
whereis 只会在标准的Linux 目录中进行搜索

选项                                 含义
-b                              只搜索可执行文件
-m                              只搜索manpage
-s                              只搜索源代码文件
-B directory                    更改或限定搜索可执行的文件的目录
-M directory                    更改或限定搜索manpage的目录
-S directory                    更改或限定搜索源代码文件的目录
```
