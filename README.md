# share-notebook
学到后面资料越难获取，不利于我们的进步，创造一个分支，分享你的笔记把！☺

1、先把你的git账号发给我，我邀请你进入仓库

2、克隆本项目

```shell
git clone git@github.com:270520006/share-notebook.git
```

3、使用命令创造属于自己的分支

```shell
git branch //查看所有分支
git branch zsp // zsp是你的分支名，自己填上自己想要的名字
```

4、切换到自己的分支

```shell
git checkout zsp
```

此后所有的笔记就在自己的分支里提交即可，更多详细资料可以看下面：
