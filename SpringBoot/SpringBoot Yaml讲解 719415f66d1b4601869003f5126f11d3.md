# SpringBoot Yaml讲解

## yaml存放对象

```yaml
sms:
  regionId: cn-hangzhou
  doMain: dysmsapi.aliyuncs.com
  version: 2017-05-25
  action: SendSms
  rid: cn-hangzhou
  singName: 富贵商城

#  存储对象行内写法
sms2: {regionId: cn-hangzhou,doMain: dysmsapi.aliyuncs.com,version: 2017-05-25,action: SendSms,rid: cn-hangzhou,singName: 富贵商城}
```

## yaml存放数组

```yaml
sms3:
  - regionId
  - accessKeyId
  - secret
# 或者
sms4: [regionId,accessKeyId,secret]
```
